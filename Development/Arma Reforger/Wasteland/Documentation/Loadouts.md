# Выбор случайного снаряжения
## Требования
1. Конфигурация должна настраиваться в Workbench
2. У каждой веши можно настраивать её "вес"
3. Есть "вес" по умолчанию
  - За счёт него вещь появляется реже или чаще
4. В конфигурации вещи распределены по зонам - ботинки, штаны, куртка, шлем, разгрузка и тп
